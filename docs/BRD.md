# BRD (Business Requirements Document)

# 안전보건 공지 현황판

**문서 버전:** 1.0
**작성일:** 2026-04-28
**소속:** 씨젠의료재단 산업보건분석팀

---

## 1. 프로젝트 배경

씨젠의료재단 산업보건분석팀은 작업환경측정 및 생물학적노출지표 검사 업무를 수행하고 있으며, 이와 관련된 안전보건 법령/규정/공지의 변경사항을 신속하게 파악하는 것이 핵심 업무 중 하나이다.

현재 관련 정보가 **안전보건공단, 산업안전보건연구원, 산업안전포털, 고용노동부** 등 여러 기관 웹사이트에 분산되어 있어, 담당자가 각 사이트를 개별적으로 방문하여 최신 공지를 확인해야 하는 비효율이 존재한다.

## 2. 비즈니스 목표

| 목표 | 설명 |
|------|------|
| **정보 접근성 향상** | 5개 기관의 최신 공지를 단일 대시보드에서 실시간 모니터링 |
| **업무 효율 개선** | 매일 여러 사이트 순회 방문 시간 절감 |
| **법령 변경 조기 파악** | 특수건강진단/작업환경측정 관련 법령 개정 및 입법예고를 즉시 인지 |
| **누락 방지** | 중요 공지의 자동 감지 및 강조 표시로 확인 누락 최소화 |

## 3. 이해관계자

| 역할 | 대상 | 관심사 |
|------|------|--------|
| **주 사용자** | 산업보건분석팀 팀장 및 팀원 | 최신 법령/규정 변경사항 파악 |
| **관련 부서** | 작업환경측정 분석담당, 생물학적노출지표 검사담당 | 업무 관련 규정 변경 추적 |
| **경영진** | 씨젠의료재단 | 법적 컴플라이언스 준수 |

## 4. 모니터링 대상 기관 및 게시판

### 4.1 안전보건공단 (KOSHA)
- **게시판:** 건설 공지사항
- **URL:** https://kosha.or.kr/notification/notice/contruction
- **중요도:** 높음 - 산업안전보건 정책 및 사업 공고

### 4.2 산업안전보건연구원 (OSHRI)
- **게시판:** 공지사항
- **URL:** https://oshri.kosha.or.kr/oshri/researcherNews/notice.do
- **중요도:** 높음 - 연구결과 발표, 채용, 업무 관련 공지

### 4.3 산업안전포털 (PORTAL)
- **게시판:** 보건위생분야 기술지원규정 (KOSHA GUIDE)
- **URL:** https://portal.kosha.or.kr/archive/resources/tech-support/search/health
- **중요도:** 매우 높음 - 보건위생 기술지원규정 제/개정 직접 관련

### 4.4 고용노동부 - 최근 제·개정 법령
- **게시판:** 최근 제·개정 법령
- **URL:** https://www.moel.go.kr/info/lawinfo/revision/list.do
- **중요도:** 매우 높음 - 산업안전보건법 등 법령 개정 사항

### 4.5 고용노동부 - 입법·행정예고
- **게시판:** 입법·행정예고
- **URL:** https://www.moel.go.kr/info/lawinfo/lawmaking/list.do
- **중요도:** 매우 높음 - 향후 시행될 법령 변경 사전 파악

## 5. 비즈니스 요구사항

### BR-01. 실시간 공지 수집
- 5개 기관 게시판에서 최신 공지 5건씩 자동 수집
- 10분 간격 자동 갱신
- 수동 새로고침 기능 제공

### BR-02. 중요 공지 강조
- 특수건강진단, 작업환경측정 관련 키워드가 포함된 공지를 **"중요"** 표시
- 대상 키워드: 특수건강진단, 작업환경측정, 산업안전보건법, 유해인자, 화학물질, 허용기준, 노출기준, 건강진단, 보건규칙

### BR-03. 신규 공지 알림
- 최근 7일 이내 게시된 공지에 **"NEW"** 뱃지 표시
- 사용자가 공지를 클릭(열람)하면 NEW 표시 제거
- 읽지 않은 신규 공지 수를 카드 헤더에 표시

### BR-04. 원문 접근
- 각 공지 클릭 시 원본 사이트의 해당 게시글로 이동
- 새 탭에서 열림

### BR-05. 상시 모니터링
- 웹 브라우저에서 상시 표시 가능한 대시보드 형태
- 다크 모드 기반 모던 Admin 스타일 UI

## 6. 성공 지표 (KPI)

| 지표 | 목표 |
|------|------|
| 일일 사이트 순회 시간 | 30분 → 5분 이내 |
| 중요 법령 변경 인지 소요시간 | 수일 → 당일 |
| 공지 확인 누락률 | 최소화 |

## 7. 제약사항

- 외부 기관 사이트의 구조 변경 시 스크래퍼 수정 필요
- 일부 사이트(KOSHA, Portal)는 SPA 기반으로 일반적인 HTML 스크래핑 불가 → API 역분석 필요
- 외부 사이트 서버 장애 시 데이터 수집 불가
- 개인정보 및 로그인이 필요한 정보는 수집 대상에서 제외
